### Абсолютный путь

**Абсолютный путь** — это путь к файлу или папке, который начинается **от корня файловой системы**.  
Он всегда однозначно указывает, где находится файл.

**Пример (Windows):**

```python
path = "C:\Users\User\Documents\project\data.txt"
```

**Пример (Linux / Mac):**

```python
path = "/home/user/project/data.txt"
```

### Относительный путь

**Относительный путь** — это путь **относительно текущей рабочей директории программы** (т.е. места, где запущен скрипт).

* Если файл лежит в той же папке, что и программа:

```python
file = "data.txt"
```

* Если файл лежит на уровень ниже (в другой папку внутри текущей директории):

```python
file = "folder/data.txt"
```

* Если нужно подняться на уровень выше:

```python
file = "../data.txt"      # ../ означает 'выйти в родительскую папку'
```
