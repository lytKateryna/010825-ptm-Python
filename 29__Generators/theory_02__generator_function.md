## Генераторная функция

**Генераторная функция** — это функция, в которой используется ключевое слово `yield`.

```python
def countdown(n):
    while n > 0:
        yield n
        n -= 1
```

Эта функция не возвращает сразу весь набор данных (как `return`),  
а **генерирует по одному значению** при каждом обращении.

---

## Создание генератор из генераторной функции

Чтобы создать генератор, нужно просто **вызвать** генераторную функцию:

```python
gen = countdown(5)
```

Теперь `gen` — это объект-генератор.
Он отдаёт значения по одному.

---

## Использование генератора

### 1. Через `next()`:

```python
print(next(gen))  # 5
print(next(gen))  # 4
print(next(gen))  # 3
```

### 2. Через цикл:

```python
for num in countdown(5):
    print(num)
```

Вывод:

```
5
4
3
2
1
```
